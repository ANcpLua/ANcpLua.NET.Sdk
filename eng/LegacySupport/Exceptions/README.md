# Exceptions

This directory provides polyfills for exception types introduced in newer .NET versions.

## Usage

To inject these exception polyfills into your project, set the following property in your `.csproj` file:

```xml
<PropertyGroup>
  <InjectANcpLuaExceptionPolyfills>true</InjectANcpLuaExceptionPolyfills>
</PropertyGroup>
```
